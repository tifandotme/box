# Box

This repository manages apps deployed in a VPS (`ssh://eddies@box.javanese-pound.ts.net`) using Kamal (v2).

Deployed apps:

- `fizzy/`
- `excalidash/`

`terraform/` provisons bucket for backup purposes.

## External

The `external/` directory contains git submodules for external apps that aren't deployed the VPS for convenience purposes. All apps aren't related.
